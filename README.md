## 🔍 Key Insights  
- Total Respondents: **1280**  
- Male: **62.5%**, Female: **37.5%**  
- Top Investment Type: **Debentures and Equity Market**  
- Main Reason for Investing: **Capital Appreciation (40%)**  
- Popular Source of Investment Information: **Internet (35%)**
